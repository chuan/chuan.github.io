3 APIs

3.1 Google Maps API

  • The radar API has 200 result limit.
  • The full text search API is not very accurate, e.g. seaching for "ATM in New York" yields about 390 results.

3.2 Factual

  • The API allows search for a given city.
  • It does not cover all the pilot cites.

3.3 OpenStreetMap (OSM)

  • In general, I like this data because it is open and in raw format. Open means anyone can contribute to the data. Raw format allows performing all sorts of queries not limited by any given APIs.
  • The API may be not very accurate in certain categories as it is created on a voluntary basis.

3.4 AngelList

  • Access requested.
  • Approved.
  • The data may be not accruate for non-western cities.

3.5 Visa/Master ATM location APIs

  • Visa and Master API need application for production use. (Sandbox API does not return real results.)
    • https://developer.visa.com/products/atmlocator/reference
    • https://developer.mastercard.com/portal/display/api/Locations
  • Visa requested.
  • Master requested.

3.6 TripAdvisor

TripAdvisor grants only a limited number of API keys, and does not allow access to the Content API for purposes of:
